CSS BASICS

CSS (Cascading Style Sheets) is a cornerstone technology used to style and layout web pages. It works alongside HTML, focusing on the presentation aspects like colors, fonts, spacing, and overall design. Understanding the basics of CSS will set you on the path to creating visually appealing websites.

CSS STYLES

  • Inline CSS: Styles written directly in HTML elements using the style attribute.
  • Internal CSS: Styles placed within 'style' tags in the 'head' of the HTML.
  • External CSS: Styles saved in a separate .css file, linked to the HTML document via a 'link' tag.
